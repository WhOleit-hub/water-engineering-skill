### 重力坝模块 — 详细操作流程

**前处理**：
1. 图形化输入断面尺寸（坝高、坝顶宽、上下游坡度）
2. 输入水位参数（正常蓄水位、设计洪水位、校核洪水位）
3. 输入材料参数（混凝土容重、抗压/抗拉强度）
4. 输入地质参数（基岩摩擦系数、黏聚力）
5. 输入地震参数（震级、地震系数）

**核心计算**：
- **应力计算**：材料力学法（截面法）+ 有限单元法，输出正应力 σ、剪应力 τ、主应力 σ₁/σ₃
- **抗滑稳定**：坝基面抗滑（纯摩法 K = f·ΣW/ΣP、剪摩法 K' = (f'·ΣW + c'·A)/ΣP）
- **深层滑动**：等K法、剩余推力法、被动抗力法，搜索最不利滑动面

**后处理**：
- 自动生成 Word 计算书（含公式、中间结果、最终结论）
- 批量生成应力云图、等值线图、位移图
- 一键生成总报告，汇总各工况（基本组合、特殊组合、地震工况）

### 水力计算模块 — 核心功能

| 计算类型 | 内容 |
|---------|------|
| 堰流计算 | 实用堰 (WES/m=0.502)、宽顶堰 (m=0.32~0.385)、薄壁堰 |
| 闸孔出流 | 平板闸门、弧形闸门，判别 e/H ≤ 0.65 |
| 水面线计算 | 棱柱体渠道（分段求和法）、非棱柱体渠道、任意断面河道（标准步推法） |
| 消能计算 | 底流（消力池深/长）、挑流（挑距/冲刷坑深）、跌水、陡坡 |
| 调洪演算 | 水库调洪（半图解法/数值积分法）、出库流量过程线 |

### 边坡稳定分析 — 详细操作

**步骤1：建立模型**
- 输入坡面线（可读入 AutoCAD DXF）
- 定义土层分层（各层 c、φ、γ 天然/饱和）

**步骤2：设置渗流条件**
- 手动输入浸润线，或从理正渗流模块导入
- 自动计算水浮力和渗透压力

**步骤3：选择计算方法**

| 方法 | 特点 | 适用条件 |
|------|------|------|
| **瑞典条分法** | 不考虑条间力，结果偏保守 | 一般圆弧滑面，校核用 |
| **简化Bishop法** | 考虑条间力，结果更精确 | 圆弧滑面，设计用 |
| **Janbu法** | 非圆弧滑面 | 有软弱夹层 |

```
瑞典条分法：K = Σ(c·l + W·cosβ·tanφ) / Σ(W·sinβ)
简化Bishop：K = Σ[(c·b + (W-u·b)·tanφ) / mβ] / Σ(W·sinβ)
其中 mβ = cosβ + tanφ·sinβ/K (迭代求解)
```

**步骤4：搜索最不利滑面**
- 指定圆心搜索区域 → 自动搜索 → 输出最小安全系数及滑面位置
- 可先粗搜（大步长）→ 再精搜（小步长），提高效率

**步骤5：结果输出**
- 计算简图（含最不利滑面）、各土条作用力
- RTF 格式计算书，可导出 DXF 图形

### 抗滑桩计算 — 关键参数

| 参数 | 说明 |
|------|------|
| **嵌固段地基方法** | M法（土层，弹性系数随深度线性变化）、C法、K法（硬土/岩层） |
| **桩内力计算模型** | 弹簧模型（桩前覆土=弹性支座，推荐）/ 荷载模型（桩前覆土=荷载） |
| **剩余下滑力** | 手动输入 或 软件从折线滑面自动计算（RT法/KT法） |
| **初始弹性系数 A/A1** | 按基坑支护规程，用于嵌固点处桩前/桩背 |

### 挡土墙设计 — 验算内容

1. **抗滑稳定**：Kc ≥ [Kc]
2. **抗倾覆稳定**：K₀ ≥ [K₀]
3. **地基承载力**：Pmax ≤ [P]、Pmin ≥ 0
4. **截面强度**：墙身任意截面拉/压应力验算
5. **整体稳定**：连同地基的圆弧滑动验算

### 水工适配要点

理正工具箱内核基于 GB 50010，用于水工结构时需注意：

1. **荷载分项系数调整**：
   - 水工规范表达式：K·S ≤ R（安全系数法）
   - 软件中调整永久荷载系数 γ_G、可变荷载系数 γ_Q，等效实现安全系数 K
   - 安全系数 K 按 SL 191 表 3.2.4 选取（一般钢筋混凝土结构 K=1.20）

2. **材料强度**：SL 191 与 GB 50010 已统一，可直接使用软件内置值

3. **计算书输出**：Word 格式，含公式、插图和中间过程

## AutoBank — 水工结构有限元分析

**研发**：河海大学工程力学研究所
**平台**：依托 AutoCAD 图形界面
**定位**：土石坝、堤防、水闸等水工建筑物的二维渗流+稳定+变形有限元分析

### 核心能力

| 分析类型 | 内容 | 理论方法 |
|---------|------|------|
| **渗流计算** | 浸润线、等势线、流速矢量、水力坡降、单宽渗流量 | 有限元法 + 达西定律 + 伽辽金加权余量法 |
| **稳定计算** | 坝坡/堤坡抗滑稳定安全系数 | 瑞典条分法 / 简化Bishop法（基于渗流计算结果） |
| **应力变形** | 应力/应变场、位移场 | 线弹性 / 邓肯-张非线性模型 |
| **固结计算** | 超静孔压消散、沉降过程 | 比奥固结理论 |

### 渗流计算 — 8步详细操作流程

```
步骤1：定义材料 → 步骤2：建立/导入模型 → 步骤3：材料分区
→ 步骤4：网格划分 → 步骤5：定义边界条件 → 步骤6：保存+更新数据
→ 步骤7：求解渗流场 → 步骤8：后处理查看结果
```

**步骤1 — 定义材料属性**
- 稳定渗流：渗透系数 Kx、Ky（可各向异性）
- 非稳定渗流：渗透系数 + **给水度**（程序可根据孔隙比用内置经验公式估算）
- 材料类型选"线弹性材料"

**步骤4 — 网格划分（最易出错环节）**

| 方法 | 方式 | 注意 |
|------|------|------|
| 半自动 | 手动选封闭区域 → 四边形单元 | 可控性好，推荐 |
| 全自动 | 定义顶点边长圆 → 三角形单元 | 需添加边长标记 |

⚠️ **网格控制要点**：
- 单元总数控制在 **1万以下**（超限报"下标越界"）
- 采用二次单元（8节点四边形/6节点三角形），精度已很高
- 不必过细划分 — 单元数从4增至256时计算结果差别很小

**步骤5 — 边界条件（求解出错的另一主因）**

| 边界类型 | 用途 | 关键操作 |
|---------|------|------|
| 固定水位边界 | 定义上/下游水位 | ⚠️ 必须用 **ESC 键**结束绘制，不能用鼠标右键！否则单宽流量=0 |
| 可能出逸边界 | 下游可能渗出面 | 沿下游边坡依次连接各拐点 |
| 水位变化边界 | 水位骤降工况 | 需指定水位-时间关系 |
| 防渗帷幕 | 模拟防渗墙/截渗墙 | 赋予相应渗透特性 |

⚠️ 水位边界线必须通过单元节点，否则浸润线与水位不衔接

**步骤6 — 保存与更新数据**
- 先保存 DWG → 再执行"更新数据"
- ⚠️ 更新后不能再移动模型，否则结果偏离

**步骤7 — 求解**
- 有限元方程：[H][φ] = [F]
- 浸润线通过节点流量平衡法迭代自动确定

**步骤8 — 后处理查看结果**

| 结果类型 | 用途 |
|---------|------|
| 浸润线 | 自由水面线位置，稳定计算的基础 |
| 等势线 | 水头等值线分布 |
| 流速矢量图 | 渗流方向和大小 |
| 水力坡降等值线 | 判断渗透破坏（距坡脚一定距离取值，非坡脚处） |
| 单宽渗流量 | 左下角显示 |

### 常见问题对策汇总

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| 单宽渗流量=0，水位=0 | 固定水位边界用鼠标右键结束 | 改用 **ESC 键**结束 |
| 浸润线与水位不衔接 | 水位边界线未通过节点 | 移动水位线附近节点至水位线处 |
| 网格划分失败 | 边界点缺边长标记 / 边长太大 | 添加边长标记 / 调小边长 |
| 查看结果"下标越界" | 单元总数 > 1万 | 增大网格尺寸 |
| 结果图形偏离模型 | "更新数据"后又移动模型 | 重新更新数据并求解 |
| 模型复制导致错误 | 同一文件多个模型 | 一个文件一个模型 |

### 水力坡降取值注意

⚠️ 理论上坡脚处水力坡降无穷大，有限元结果由高斯积分点平均得到。工程中应**取距坡脚一定距离处的水力坡降**作为渗透破坏评判依据。

### 典型工作流组合

```
土石坝设计：
  AutoBank 渗流计算(浸润线) → AutoBank 坝坡稳定(安全系数)
  → 远盛水工 开挖+土方 → 理正结构 溢洪道配筋

重力坝设计：
  远盛水工 应力+稳定(各工况) → 理正结构 廊道/孔口配筋

水闸设计：
  远盛水工 水闸整体稳定+渗流 → 理正结构 底板/闸墩/翼墙配筋
  → 理正岩土 岸墙/翼墙挡土墙验算

堤防/河道：
  AutoBank 渗流+稳定断面 → 远盛水工 开挖断面+工程量
  → 理正岩土 岸坡稳定
```

| 列 | 说明 |
|:--|:-----|
| river | 河道名 |
| reach | 河段名 |
| RS | 桩号（同一断面的所有点RS相同） |
| station | 起点距(m) |

注意事项：
- 每行一个测点，同一断面的所有点必须有相同RS值
- 先画河道中心线（River工具）再导入
- 导入后需手动补Manning's n、Bank Sta、Exp/Cntr等参数
- HEC-RAS 7.x 不支持 `,,,` 空列尾

### Manning's n 在 .g01 中的格式

HEC-RAS 7.x 使用固定8字符列宽：

```
#Mann= 3 ,0,0
  左岸sta    .nnn       0       0左岸sta    .nnn       0  右岸sta    .nnn       0
```

每值8字符宽，9个值共72字符。3组breakpoint分别对应左滩、主槽、右滩。

### HEC-RAS Mixed Flow 模式

适用场景：同一河段既有缓流又有急流（如山洪沟）。
设置：Run → Steady Flow Analysis → Flow Regime → 选 Mixed
- 亚临界边界设下游（已知水位或正常水深）
- 超临界边界设上游
- HEC-RAS 自动处理水跃位置
