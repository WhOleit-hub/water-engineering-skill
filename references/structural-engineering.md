## 3B. 水工钢筋混凝土结构

> **依据**：SL 191（水工混凝土结构）、DL/T 5057
> **教材**：汪基伟《水工钢筋混凝土结构学》第5版（四校合编）

### 3B.1 教材体系概览

**12章 + 11附录**，河海/武大/大连理工/郑州大学四校合编，**SL 191-2008 与 DL/T 5057-2009 双规范对照**

### 3B.2 水工混凝土结构设计表达式

两种规范体系并存，区别与换算关系：

| 规范 | 表达式 | 说明 |
|------|------|------|
| **SL 191** | $K \cdot S \leq R$ | 安全系数法：K—安全系数；S—荷载效应；R—结构抗力 |
| **DL/T 5057** | $\gamma_0 \psi S \leq R/\gamma_d$ | 分项系数法：γ₀/ψ/γ_d — 结构重要性/设计状况/结构系数 |

> **转换关系**：$K = \gamma_0 \cdot \psi \cdot \gamma_G \cdot \gamma_Q \cdot \gamma_d \approx \gamma_0 \cdot \psi \cdot \gamma_d$（永久荷载为主时）
> SL 191 安全系数 K 查表 3.2.4（一般钢筋砼受弯 K=1.20）

### 3B.3 核心计算模块

#### 受弯构件正截面（第3章）

| 截面类型 | 适用条件 |
|------|------|
| 单筋矩形 | $\xi = x/h_0 \leq 0.85\xi_b$（防止超筋）；$\rho \geq \rho_{min}$（防止少筋） |
| 双筋矩形 | 当单筋不满足 ξ 限制时使用 |
| T 形截面 | 第Ⅰ类（中和轴在翼缘内，$x \leq h_f'$）；第Ⅱ类（中和轴在腹板） |

**承载力公式**（单筋矩形）：

$$\alpha_1 f_c b x = f_y A_s \quad\text{(力平衡)}$$
$$M_u = \alpha_1 f_c b x (h_0 - x/2) = f_y A_s (h_0 - x/2) \quad\text{(力矩平衡)}$$

#### 受弯构件斜截面（第4章）

| 破坏形态 | 特征 |
|------|------|
| 斜压破坏 | 剪跨比 λ 小，腹筋过多 |
| 剪压破坏 | λ 适中，正常设计 |
| 斜拉破坏 | λ 大，腹筋过少 |

#### 受压构件（第5章）

- 轴心受压：$N \leq \varphi (f_c A + f_y' A_s')$
- 偏心受压：大偏心（受拉破坏，ξ≤ξ_b）vs 小偏心（受压破坏，ξ>ξ_b）

### 3B.4 水工特殊问题（第12章）

| 问题 | 要点 |
|------|------|
| **最小配筋率** | SL 191 规定水工大体积混凝土最小配筋率（不同于 GB 50010） |
| **温度配筋** | 大体积混凝土温度作用下的抗裂验算与温度钢筋配置 |
| **非杆件体系配筋** | 按应力图形法进行配筋设计 |

### 3B.5 正常使用极限状态（第8章）

| 验算项 | 控制指标 |
|------|------|
| 抗裂 | 一级（严格要求不裂）：$\sigma_{ck} - \sigma_{pc} \leq 0$；二级（一般要求不裂）：$\sigma_{ck} - \sigma_{pc} \leq f_{tk}$ |
| 裂缝宽度 | $w_{max} \leq [w]$（水工 $[w]=0.20\sim0.30mm$） |
| 挠度 | $f \leq [f]$ |

### 3C.2 平面钢闸门设计（第7章 — 核心）

**组成**：面板 + 次梁（水平/竖直）+ 主梁 + 边梁 + 横向联结系 + 纵向联结系

**10节完整设计流程**：

```
§7.1 概述（闸门类型/工作条件）
§7.2 结构布置（主梁/次梁/联结系布置原则）
§7.3 面板和次梁设计 → 面板厚度 δ = a√(σ_my/σ_cr)
§7.4 主梁设计 → 弯矩/剪力包络图 → 截面选择/验算
§7.5 横向联结系 → 保证闸门整体空间刚度
§7.6 边梁设计 → 传递水压力到支承
§7.7 行走支承 → 滑动/滚动/定轮支承
§7.8 轨道及埋件 → 闸槽/底坎/侧轨
§7.9 止水 + 启闭力 + 吊耳
§7.10 设计例题 — 露顶式平面钢闸门完整设计
```

**闸门启闭力**：
$$F_T = n_T (T_{zd} + T_{zs}) + n_G G + P_t$$

其中 $T_{zd}$ 为支承摩阻力，$T_{zs}$ 为止水摩阻力，G 为闸门自重，$P_t$ 为上托力

### 3C.3 连接设计要点（第3章）

| 连接方式 | 适用场景 | 关键公式 |
|------|------|------|
| 对接焊缝 | 主梁腹板/翼缘拼接 | $\sigma = N/(l_w t) \leq f_t^w$ |
| 角焊缝 | 次梁与面板、加劲肋 | $\tau_f = N/(h_e \sum l_w) \leq f_f^w$ |
| 普通螺栓 | 安装连接/可拆卸部位 | $N_v^b = n_v \cdot \pi d^2/4 \cdot f_v^b$ |
| 高强螺栓 | 主梁工地拼接 | 摩擦型：$N_v^b = 0.9 n_f \mu P$ |
