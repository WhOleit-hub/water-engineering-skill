## 0. 水利分析第一原则：先读图，再计算

> ⚠️ **铁律**：任何水利计算开始前，必须先在图上跑一遍水流路径。不看图不计算。

### 为什么

错误的水流路径 → 错误的上下游边界 → 错误的数据来源 → 计算全错

**实战教训**（旗山湖浮桥洪评）：未看流域布局图，从文字推断出水口为侯官水闸。实际旗山湖出流路径是旗山湖→旗山湖节制闸→轮船港→厚庭水闸→闽江。侯官水闸位于溪源江干流另一条线上（邱阳河口），不控制旗山湖出流。**用错了泄流曲线，整节推导重做。**

### 强制流程

```
Step 1: 打开流域布局图 / 排涝分区图 / 水系图
Step 2: 沿水流方向，从上游到下游逐节点确认：
        ┌─ 来水：哪些河流/渠道汇入？
        ├─ 控制工程：沿途有哪些闸/坝/泵站/水库？
        ├─ 出水口：水最终从哪里排出（哪个闸/泵站）？
        └─ 关联工程：其他可能影响水流的工程？
Step 3: 确认项目所在位置在流域中的水力关系
Step 4: 在图上标出水流路径（上游→项目→下游出口）
Step 5: 根据确认的路径选择正确的计算数据
Step 6: 最后才开始计算
```

### 常见踩坑

| 坑 | 案例 |
|------|------|
| 一个流域有多个出口闸，搞混 | 旗山湖→厚庭闸，不是侯官闸 |
| 同名不同物（闸vs泵站） | 侯官水闸≠侯官泵站，功能完全不同 |
| 图上看着近，水力上不连通 | 隔着一道节制闸就是两条独立水路 |
| 上/下册报告覆盖不同片区 | 下册是南屿片区，上册才是大学城片区 |
| 扩建的是泵站不是水闸 | 泵站扩建（强排能力提升）≠水闸扩建（自排能力提升） |

## 1. 标准规范体系

计算和设计时必须优先参考中国标准，而非国际规范。中国水利标准分两层：

- **GB / GB/T** — 国家标准（强制性 / 推荐性）
- **SL / SL/T** — 水利行业标准（2024年起全面转为推荐性 SL/T）

⚠️ 注意标准时效：大量 2000s~2010s 标准已在 2024-2026 年修订，优先使用最新版本。

### 1.4 标准使用原则

1. **GB 强制性** > GB/T 推荐性 > SL/T 推荐性

## 2. 水文分析与计算

> **依据**：SL/T 44（设计洪水）、SL/T 278（水文计算）、GB/T 50095（水文术语）、GB/T 22482（水文情报预报）
> **教材**：詹道江《工程水文学》+ 梁忠民《水文分析与计算》

### 2.1 水文循环与径流形成

**水文循环**：海洋→水汽输送→降水→径流（地表/地下）→蒸发→回归

**径流形成过程**：
```
降水 → 植物截留 → 填洼 → 下渗 → 产流（超渗产流/蓄满产流）
     → 坡面汇流 → 河网汇流 → 流域出口断面流量过程线
```

**水量平衡方程**（任意时段）：$P = R + E + \Delta W$

式中：P—降水量；R—径流量；E—蒸发量；ΔW—蓄水变量

### 2.2 水文统计与 P-III 型频率曲线

中国水文频率分析必须使用 **P-III 型分布**（不是 Gumbel 或 GEV）。

**三参数**：均值 $\bar{x}$、变差系数 $C_v$、偏态系数 $C_s$

$$x_p = \bar{x}(1 + C_v \cdot \Phi_p) = \bar{x} \cdot K_p$$

离均系数 $\Phi_p$ 通过 P-III 分布 Φ 值表查取（由 C_s 和设计频率 p 确定）。
模比系数 $K_p = 1 + C_v \cdot \Phi_p$

**参数估计**：

| 参数 | 公式 | 说明 |
|------|------|------|
| 均值 | $\bar{x} = \frac{1}{n}\sum x_i$ | 算术平均 |
| C_v | $C_v = \sqrt{\frac{\sum(K_i-1)^2}{n-1}}$ | K_i = x_i/$\bar{x}$ |
| C_s | 矩法初步估计 + 适线调整 | C_s/C_v 一般取 2.0~4.0 |

**适线法**（经验适线法）：

1. 将实测系列从大到小排列，计算经验频率 $p = m/(n+1)$
2. 点绘 (x_i, p_i) 于**海森机率格纸**
3. 初选 $\bar{x}$、C_v、C_s，计算理论频率曲线
4. 目估比较理论与经验点据 → 调整参数 → 适线最佳（侧重中上部点据）
5. 确定采用参数 → 计算各频率设计值

> ⚠️ 目估适线是中国规范规定方法，优先于纯统计拟合（纯矩法/极大似然法不符合中国规范）

**不连序系列处理**：含历史洪水（考证期 N > 实测期 n）
- 经验频率：实测洪水用 $p=m/(n+1)$，历史洪水用 $p=M/(N+1)$
- 统计参数用加权矩法计算

### 2.3 由流量资料推求设计洪水

**步骤**：
1. 年最大选样（年最大值法）— 洪峰 Q_m、各时段洪量 W_t（1d/3d/7d/15d/30d）
2. 加入历史洪水 → 不连序系列
3. P-III 频率计算 → 各频率设计洪峰、设计洪量
4. 选择典型洪水过程线（峰高量大、对工程不利）
5. 同倍比放大法或同频率放大法 → 设计洪水过程线

**放大方法**：

| 方法 | 原理 | 适用 |
|------|------|------|
| 同倍比放大 | 按 Q_{mp}/Q_{m典} 比例统一放大 | 峰量关系好，单峰型 |
| 同频率放大（分时段） | 洪峰和各时段洪量分别按各频率设计值控制放大 | 峰量关系差，规范推荐 |

### 2.4 由暴雨推求设计洪水（无资料地区）

**设计暴雨计算**：
- 点暴雨：P-III 频率分析 → 各历时设计点暴雨量 (1h/6h/24h/3d)
- 面暴雨：$H_{面} = \alpha \cdot H_{点}$（α 点面折减系数，按流域面积查各省水文手册）
- 暴雨时程分配：采用当地典型雨型或概化雨型分配

**产流计算**：

| 方法 | 原理 | 适用地区 |
|------|------|------|
| 降雨径流相关法（P+Pa~R） | 经验相关图 | 资料条件好 |
| 初损后损法 | 初损 I₀（产前损失）+ 后损 $\bar{f}$（平均下渗率） | 超渗产流区（北方） |
| 蓄满产流模型 | 田间持水量 W_m 控制，R = P - (W_m - W₀) | 湿润地区 |

**汇流计算**：

| 方法 | 公式 | 适用 |
|------|------|------|
| 单位线法 | $Q(t) = \sum R_i \cdot u(t-i+1)$ | 有实测资料 |
| 瞬时单位线（Nash） | $u(t) = \frac{1}{K\Gamma(n)}(t/K)^{n-1}e^{-t/K}$ | 无资料→参数综合 |
| 推理公式法 | $Q_m = 0.278 \psi \cdot S \cdot F / \tau^n$ | 小流域（F<500km²） |

### 2.5 河道洪水演进 — 马斯京根法

$$O_2 = C_0 I_2 + C_1 I_1 + C_2 O_1$$

$$C_0 = \frac{0.5\Delta t - Kx}{K - Kx + 0.5\Delta t},\quad C_1 = \frac{0.5\Delta t + Kx}{K - Kx + 0.5\Delta t},\quad C_2 = \frac{K - Kx - 0.5\Delta t}{K - Kx + 0.5\Delta t}$$

约束：C₀+C₁+C₂=1；K 为传播时间，x 为流量比重因子（0~0.5）

### 2.6 水库兴利调节计算

**水库特性**：特征水位（死水位/正常蓄水位/防洪限制水位/设计洪水位/校核洪水位）
特征库容（死库容/兴利库容/防洪库容/总库容）

**兴利调节**（依据周之豪《水资源规划及利用》）：
- 时历法（长系列操作）：列表逐年逐月进行水量平衡
  $$V_{t+1} = V_t + W_{来} - W_{用} - W_{损}$$
- 典型年法：选设计保证率典型年 → 确定兴利库容

### 2.7 水文预报

| 方法 | 内容 |
|------|------|
| 相应水位法 | 上下游洪峰水位经验相关 |
| 降雨径流预报 | P+Pa~R + 单位线 → 预报流量过程 |
| 实时校正 | 卡尔曼滤波/误差自回归校正

## 3. 水力学

> **依据**：SL/T 253（溢洪道）、SL/T 265（水闸）、SL 252（等级划分及洪水标准）、SL/T 278（水文计算）
> **教材**：吴持恭《水力学》第4版

### 3.1 明渠恒定均匀流

**基本公式 — 谢才公式与曼宁公式：**

$$Q = AC\sqrt{RJ} = \frac{1}{n} A R^{2/3} J^{1/2}$$

式中：A — 过水断面面积；R — 水力半径 (R=A/χ，χ为湿周)；J — 底坡（均匀流时 J=i）；n — 糙率

**正常水深 h₀**：均匀流时的水深，满足 $Q = \frac{1}{n} A R^{2/3} \sqrt{i}$

> 矩形断面：$Q = \frac{1}{n} b h_0 \cdot \left(\frac{b h_0}{b + 2h_0}\right)^{2/3} \sqrt{i}$，需迭代求解

**临界水深 hₖ**：断面比能最小时的水深，满足：

$$\frac{Q^2}{g} = \frac{A_k^3}{B_k}$$

> 矩形断面简化：$h_k = \sqrt[3]{\frac{q^2}{g}}$，其中 $q = Q/b$（单宽流量）

| 底坡分类 | 条件 | 水面线类型 |
|:--:|------|:--:|
| 陡坡 (Steep) | i > iₖ (h₀ < hₖ) | S1, S2, S3 |
| 缓坡 (Mild) | i < iₖ (h₀ > hₖ) | **M1, M2, M3** |
| 临界坡 (Critical) | i = iₖ (h₀ = hₖ) | C1, C3 |
| 平坡 (Horizontal) | i = 0 | H2, H3 |
| 逆坡 (Adverse) | i < 0 | A2, A3 |

**水力最佳断面**：给定面积时过流量最大的断面形状。

> 梯形断面最佳宽深比：$\beta_m = \frac{b}{h} = 2(\sqrt{1+m^2} - m)$（m 为边坡系数）

### 3.2 明渠恒定非均匀流 — 水面线计算

**水面曲线微分方程**（棱柱体渠道）：

$$\frac{dh}{ds} = \frac{i - \frac{Q^2}{K^2}}{1 - Fr^2} = \frac{i - J}{1 - Fr^2}$$

式中 Fr = v/√(gh)（弗劳德数），K = A·C·R^(1/2)（流量模数）

**逐段试算法**（标准步推法）— 适用于棱柱体和非棱柱体渠道：

$$\Delta s = \frac{E_{s2} - E_{s1}}{i - \bar{J}} = \frac{(h_2 + \frac{v_2^2}{2g}) - (h_1 + \frac{v_1^2}{2g})}{i - \bar{J}}$$

式中 $\bar{J} = \frac{J_1 + J_2}{2} = \frac{1}{2}\left(\frac{Q^2}{K_1^2} + \frac{Q^2}{K_2^2}\right)$

**计算步骤**：
1. 从控制断面（已知水深）出发
2. 假定相邻断面水深 → 计算 Δs
3. 若 Δs 不等于已知断面间距 → 修正水深重算
4. 逐段推进至全渠段

**常见水面线（缓坡 M 型）**：

| 类型 | 特征 | 出现位置 |
|:--:|------|------|
| **M1** | h > h₀ > hₖ，壅水曲线 | 闸坝上游回水区 |
| **M2** | h₀ > h > hₖ，降水曲线 | 渠道末端跌水前 |
| **M3** | h₀ > hₖ > h，壅水曲线 | 闸后/陡坡下游 |

### 3.3 水跃

**水跃方程**（平底棱柱体渠道）：

$$\frac{Q^2}{gA_1} + A_1 h_{c1} = \frac{Q^2}{gA_2} + A_2 h_{c2}$$

矩形断面简化 → **共轭水深公式**：

$$h'' = \frac{h'}{2}\left(\sqrt{1 + 8Fr_1^2} - 1\right)$$

**水跃长度**（矩形渠道经验公式）：
$$L_j = 6.9(h'' - h') \quad \text{（欧勒佛托斯基）}$$

**水跃消能率**：
$$\frac{\Delta E}{E_1} = \frac{(\sqrt{1+8Fr_1^2} - 3)^3}{8(\sqrt{1+8Fr_1^2} - 1)(Fr_1^2 + 2)}$$

### 3.4 堰流（Weir Flow）

**堰流基本公式**（吴持恭《水力学》）：

$$Q = \sigma_s \cdot \varepsilon \cdot m \cdot n b \sqrt{2g} \cdot H_0^{3/2}$$

| 参数 | 名称 | 说明 |
|------|------|------|
| m | 堰流流量系数 | 由堰型决定 |
| ε | 侧收缩系数 | ε=1 无侧收缩 |
| σₛ | 淹没系数 | σₛ=1 自由出流 |
| n | 孔数 | — |
| b | 单孔净宽 | m |
| H₀ | 堰顶总水头 | H₀=H+v₀²/2g |

**堰型判别**（δ为堰顶厚度）：

| 类型 | δ/H 范围 | 典型应用 |
|------|:--:|------|
| 薄壁堰 | δ/H < 0.67 | 流量测验 |
| 实用堰 | 0.67 < δ/H < 2.5 | 溢洪道/溢流坝 |
| 宽顶堰 | 2.5 < δ/H < 10 | 水闸/桥涵/涵洞 |

#### 3.4.1 WES 实用堰

**设计水头**：$H_d = (0.75 \sim 0.95) H_{max}$

**高堰**（P₁/H_d ≥ 1.33）设计流量系数：**m_d = 0.502**（H₀=H_d 时）

当 H₀ ≠ H_d 时：m = m_d × (m/m_d)，m/m_d 由试验曲线查取。

- H₀ > H_d → 堰面出现负压，m > 0.502
- H₀ < H_d → 堰面压力增大，m < 0.502

**低堰**（P₁/H_d < 1.33）：$m_d = 0.4987(P_1/H_d)^{0.0241}$

$$\varepsilon = 1 - 0.2[\xi_k + (n-1)\xi_0]\frac{H_0}{nb} \quad (H_0/b \leq 1.0)$$

## 3A. 水工挡土墙设计

> **依据**：SL 379（挡土墙）、SL 274（土石坝）、SL 265（水闸）、SL 203（抗震）
> **教材**：管枫年《水工挡土墙设计》+ 顾慰慈《土压力和挡土墙计算简明手册》

### 3A.1 结构型式

| 型式 | 适用条件 | 特点 |
|------|------|------|
| **重力式** | 墙高 ≤ 5m | 靠自重稳定，施工简单，仰斜/垂直/俯斜/衡重式 |
| **衡重式** | 墙高 ≥ 5m | 设衡重台，上墙利用第二破裂面减载 |
| **悬臂式** | 墙高 ≤ 8m | 由立壁+底板组成，钢筋混凝土 |
| **扶壁式** | 墙高 ≥ 8m | 悬臂式加扶壁，钢筋混凝土 |
| **板桩式** | 地基软弱 | 板桩挡土 |
| **加筋土** | 填方较高 | 柔性结构，地基适应性好 |

### 3A.2 土压力计算

#### 库仑理论（Coulomb，1773）

适用：砂性土、墙背倾斜粗糙、填土面不规则

$$E_a = \frac{1}{2}\gamma H^2 K_a$$

$$K_a = \frac{\cos^2(\varphi - \alpha)}{\cos^2\alpha \cdot \cos(\alpha + \delta)\left[1 + \sqrt{\frac{\sin(\varphi + \delta)\sin(\varphi - \beta)}{\cos(\alpha + \delta)\cos(\alpha - \beta)}}\right]^2}$$

| 参数 | 含义 | 典型值 |
|------|------|:--:|
| α | 墙背与铅垂线夹角 | 俯斜为正，仰斜为负 |
| φ | 填土内摩擦角 | 砂性土 28°~35° |
| β | 填土面与水平面夹角 | — |
| δ | 土与墙背外摩擦角 | δ = φ/3 ~ 2φ/3 |

#### 朗肯理论（Rankine，1857）

适用：墙背竖直光滑（δ=0）、填土面水平（β=0）

$$E_a = \frac{1}{2}\gamma H^2 K_a - 2cH\sqrt{K_a}$$

$$K_a = \tan^2(45^\circ - \frac{\varphi}{2})$$

| 对比 | 库仑 | 朗肯 |
|:--|:--|:--|
| 适用范围 | 广（可不同坡度/粗糙度） | 窄（需墙背竖直光滑） |
| 砂性土 | ✓ | ✓ |
| 粘性土 | 需等效内摩擦角 φ_d | ✓（含 c 项） |
| 主动土压力 | 较准确 | 偏大（保守） |
| 被动土压力 | 误差大（不推荐） | 偏小（保守，推荐） |

#### 第二破裂面法（坦墙理论）

**出现条件**：墙背倾角 α 过大（坦墙）

$$\alpha_{cr} = 45^\circ - \frac{\varphi}{2} + \frac{\beta}{2} - \frac{1}{2}\arcsin\left(\frac{\sin\beta}{\sin\varphi}\right)$$

当 β=0：$\alpha_{cr} = 45^\circ - \varphi/2$

当 α > α_cr 时，出现第二破裂面（土体与土体之间滑动，非土体与墙背）。

**典型应用**：衡重式挡土墙上墙、L 型墙背（悬臂式/扶壁式）

### 3A.3 稳定验算

#### 抗滑稳定

$$K_c = \frac{\sum V \cdot f}{\sum H} \geq [K_c]$$

| 地基 | [Kc] |
|:--|:--:|
| 土基 | 1.25~1.30 |
| 岩基 | 1.05~1.10 |

f — 基底摩擦系数（土基 0.30~0.45，岩基 0.50~0.70）

**增加抗滑措施**：设置倾斜基底（坡度 1:5~1:3）、采用凸榫基础

#### 抗倾覆稳定

$$K_t = \frac{\sum M_V}{\sum M_O} = \frac{W \cdot a + P_y \cdot b}{P_x \cdot H/3} \geq 1.50$$

#### 基底应力与偏心距

$$P_{max/min} = \frac{\sum V}{B}\left(1 \pm \frac{6e}{B}\right)$$

$$e = \frac{B}{2} - \frac{\sum M_{墙趾}}{\sum V}$$

| 验算项 | 要求 |
|:--|:--|
| 偏心距 e | ≤ B/6（土基）；≤ B/5（岩基） |
| 平均应力 | P_avg ≤ [R]（地基承载力） |
| 最大应力 | P_max ≤ 1.2[R] |
| 应力比 η | η = P_max/P_min ≤ **3.0**（土基） |

### 3D.1 重力坝

**设计内容**：断面拟定 → 荷载计算 → 应力分析 → 抗滑稳定 → 泄水孔口设计

**荷载组合**：基本组合（正常蓄水位+自重+扬压力+淤沙+浪压力）→ 特殊组合（校核水位/地震）

**抗滑稳定**：
- 坝基面：纯摩 $K = f \cdot \sum W / \sum P$；剪摩 $K' = (f' \sum W + c'A) / \sum P$
- 深层滑动：等K法/剩余推力法/被动抗力法搜索最危险滑动面

**应力计算**：材料力学法（σ = ΣW/B ± 6ΣM/B²）→ 允许压应力取决于混凝土标号，不允许出现拉应力或控制拉应力范围

### 3D.2 土石坝

**坝型**：均质坝/心墙坝/斜墙坝/面板堆石坝

**渗流计算**：
- 浸润线：分段法（上游三角形段+中间段+下游段），水力学解法
- 单宽渗流量：$q = k \cdot \frac{H_1^2 - h_k^2}{2L}$
- 出逸点高度 $h_0$ 需迭代求解

**坝坡稳定**：瑞典圆弧法/简化Bishop法，规范要求安全系数 ≥ 1.15~1.50（按工程等级和工况）

### 3D.3 水闸

**整体稳定**：抗滑/抗倾/抗浮/地基承载力/沉降量

**防渗设计**：改进阻力系数法或流网法 → 确定地下轮廓线（铺盖+板桩+排水）

**消能防冲**：底流消能（消力池深d、池长L_k），海漫+防冲槽

> ```python
> from scipy import stats
> def p3_quantile(p, mean, Cv, Cs):
>     alpha = 4 / Cs**2
>     beta = 2 / (mean * Cv * Cs)
>     loc = mean * (1 - 2*Cv/Cs)
>     return stats.gamma.ppf(1-p/100, alpha, loc=loc, scale=1/beta)
> 
> # 例：计算百年一遇设计洪峰
> Q_design = p3_quantile(1, mean=1250, Cv=0.45, Cs=1.80)  # Q1%=? m³/s
> ```
>
> 推理公式法（无资料小流域洪峰）用以下模板：
> ```python
> def rational_formula(S, n, F, L, J, m, mu):
>     """推理公式法 Qm = 0.278·ψ·S·F/τⁿ（迭代求解）"""
>     Qm = 100  # 初始假定
>     for _ in range(30):
>         tau = 0.278 * L / (m * J**(1/3)) / Qm**0.25
>         psi = 1 - mu / (S * tau**n) if tau <= 1 else n*(tau-1)/tau + 1 - mu/(S*tau)
>         Qm_new = 0.278 * psi * S * F / tau**n
>         if abs(Qm_new - Qm) < 0.01: break
>         Qm = Qm_new
>     return Qm
> ```

### 3.8 陡坡水面线计算注意事项

山洪沟（坡度 > 2%）水面线计算有特殊问题需要处理：

**能量方程双解问题**：同一个总能量 E = z + h + V²/(2g) 对应两个水深
- 缓流解 Fr<1：水深大、流速小（物理上正确）
- 急流解 Fr>1：水深小、流速极大（求解器易收敛至此）

**处理方法**：
1. 加 Fr < 1 约束：搜索时跳过 Fr > 0.95 的解
2. 从深水向下搜索（h=5m→0.1m），而非从浅水向上
3. 计算临界水深 hc 并约束 h > hc
4. 急流段从上游往下游算，缓流段从下游往上游算，水跃处连接
5. 最根本方案：使用 HEC-RAS Mixed Flow 模式

**简化方案**：陡坡山洪沟可用正常水深法分段近似
